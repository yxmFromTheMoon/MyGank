# MyGank
基于代码家数据的Gank客户端

### 闲来无事，随便写写，最近看到代码家api升级了，以前也没用过，正好拿来学习一下androidx和viewpager2
### 整体架构MVC + Retrofit + OkHttp + Glide，之前写过一个MVP架构的，所以这次就没怎么特别在意架构。
### 后续想用MVVM来重构一下
